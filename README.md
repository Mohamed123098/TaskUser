# 📌 TaskUser - Angular Task Management App

A modern task management web application built with Angular that allows users to manage their daily tasks efficiently.

---

## 🚀 Features

- ✅ Add new User  
- 🗑️ Delete tasks  
- 📋 Display task list   
- ⚡ Responsive UI  

---

## 🛠️ Technologies Used

- Angular  
- TypeScript  
- HTML & tailWindCSS  
- RxJS   

---

## ⚙️ Prerequisites

Make sure you have installed:

- Node.js (v24 or higher)
- npm
- Angular CLI

Install Angular CLI:

```bash
npm install -g @angular/cli
```

---

## 📥 Installation

Clone the repository:

```bash
git clone https://github.com/Mohamed123098/TaskUser.git
```

Navigate to the project:

```bash
cd TaskUser
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Run the Application

Start the development server:

```bash
ng serve --open
```
```OR
ng s
```
Open your browser at:

```
http://localhost:4200/
```
